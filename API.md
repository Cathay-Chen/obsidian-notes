    $(function() {

# 主要特性

-   支持“标准” Markdown / CommonMark 和 Github 风格的语法，也可变身为代码编辑器；
-   支持实时预览、[图片（跨域）上传](http://editor.md.ipandao.com/examples/image-upload.html)、预格式文本/代码/表格插入、代码折叠、搜索替换、只读模式、自定义样式主题和多语言语法高亮等功能；
-   支持 [ToC 目录](http://editor.md.ipandao.com/examples/toc.html)（Table of Contents）、[Emoji 表情](http://editor.md.ipandao.com/examples/emoji.html)、[Task lists](http://editor.md.ipandao.com/examples/task-lists.html)、[@链接](http://editor.md.ipandao.com/examples/@links.html)等 Markdown 扩展语法；
-   支持 [TeX 科学公式](http://editor.md.ipandao.com/examples/katex.html)（基于 KaTeX）、[流程图](http://editor.md.ipandao.com/examples/flowchart.html) Flowchart 和 [时序图](http://editor.md.ipandao.com/examples/sequence-diagram.html) Sequence Diagram;
-   支持识别和解析 HTML 标签，并且支持自定义过滤标签解析，具有可靠的安全性和几乎无限的扩展性；
-   支持 AMD / CMD 模块化加载（支持 Require.js & Sea.js），并且支持自定义扩展插件；
-   兼容主流的浏览器（IE8+）和 Zepto.js，且支持 iPad 等平板设备；
-   支持自定义主题样式；
